O que são Hooks?

  • Recurso do React que tem diversas funções.
  • Como: guardar e alterar o estado de algun dado na nossa aplicação.
  • Todos os hooks começam com use, por eemplo useState.
  • Podemos criar os nossos hooks, isso é chamado de custom hook.
  • Os hooks precisam ser importados.
  • Geralmente são úteis em todas as aplicações, utilizamos diversos ao longo do curso.

useState

  • O hook de useState é um dos mais utilizados.
  • Utilizamos para gerenciar o estado de algum dado, variáveis não corretamente, o componente não re-renderiza.
  • Para guardar o dado definimos o nome da variável e para alterar vamos utilizar o setNoma, onde nome é o nome do nosso dado.
  • Vamos ver na prática!
