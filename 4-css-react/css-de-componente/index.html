CSS de componente

  • O css de componente é utilizado para um componente em especifico
  • Geralmente é criado um arquivo com o mesmo nome do componente e este é importado no componente
  • Note que este método não é scoped, ouseja, o CSS vaza para outros componente se houver uma regra de colisão
  • O react ja cria um exemplo desta técnica com o App.css/js
